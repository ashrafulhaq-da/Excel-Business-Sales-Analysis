📊 Business Sales Analysis Dashboard (Excel Project)

 📌 Overview
This project analyzes retail business sales data to generate insights into revenue, profitability, customer behavior, and operational performance using Microsoft Excel.

---

🎯 Objectives
- Analyze sales performance across products and regions
- Identify high-revenue and high-profit segments
- Understand purchasing patterns
- Build a professional dashboard for decision-making

---

 📂 Dataset Details
The dataset includes:

- Order ID
- Date
- Product Category
- Price
- Quantity
- Cost
- Purchase Type (Online/In-store)
- Payment Method
- Manager
- City & Country
- Revenue
- Profitability



🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning Functions
- Dashboard Design



🔍 Data Cleaning Process
- Removed extra spaces using TRIM & CLEAN
- Fixed inconsistent manager names
- Standardized columns
- Validated numerical fields



📊 Analysis Performed

1. Revenue Analysis
- Total Revenue by Product Category
- Revenue by Country and City

2. Profitability Analysis
- Profit comparison across regions
- High vs low margin products

3. Sales Behavior
- Online vs In-store purchase trends
- Payment method distribution

 4. Manager Performance
- Sales handled by each manager
- Revenue contribution per manager
  

📈 Key Insights
- Groceries and Household Items generate the highest revenue
- Certain cities (e.g., Berlin, London) dominate total sales
- Online purchases are slightly higher than in-store
- Credit Card is the most used payment method
- Some managers consistently outperform others in revenue generation



📊 Dashboard Features
- Interactive Pivot Tables
- Dynamic Charts
- KPI Summary (Revenue, Profit, Orders)
- Clean and professional layout

This project showcases how Excel can be used as a powerful data analysis tool to extract business insights and support decision-making.
